FAQ Accordion Component

Small, accessible accordion built with semantic HTML as part of the HTML/CSS/JS mini projects portfolio.

This is a small FAQ accordion component built with semantic HTML, modern CSS, and vanilla JavaScript. It focuses on accessibility and clean structure, making it a good practice project for frontend fundamentals.

Each question is a button with proper ARIA attributes. Clicking toggles aria-expanded and a CSS class on the item to show or hide the corresponding answer panel, while keeping the markup accessible for keyboard and screen reader users.

Proper semantics and ARIA attributes ensure that keyboard users and screen reader users can understand and operate the accordion, not just mouse users.

Built as part of the html-css-js-portfolio project.
  • Semantic HTML
  • ARIA-friendly accordion
  • Keyboard & screen reader support
