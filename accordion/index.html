FAQ Accordion Component

Small, accessible accordion built with semantic HTML as part of the HTML/CSS/JS mini projects portfolio.

This is a learning project that focuses on building an accessible FAQ accordion using only HTML first, then adding CSS and JavaScript.

JavaScript will toggle the visibility of each answer and update the aria-expanded attribute on the buttons so that screen readers know which section is open.

Proper semantics and ARIA attributes ensure that keyboard users and screen reader users can understand and operate the accordion, not just mouse users.
