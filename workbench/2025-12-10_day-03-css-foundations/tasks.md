# ✅ Day 03 Tasks – Responsive Nav

## Učenje

- [ ] Pročitaj novi `navigation.css` i zapiši šta se menja na 768px i 1024px.
- [ ] Pogledaj `index.css` nav blokove i inline media queries.
- [ ] Razumi zašto koristimo `overflow-x: auto` + `white-space: nowrap`.

## Praksa

- [ ] Testiraj nav u DevTools na 320, 375, 425, 768, 900, 1024, 1280.
- [ ] Ako nešto puca na 900–1020px, prilagodi gap/padding ili dodaj max-width headera.
- [ ] Proveri fokus state (Tab) i hover na svim veličinama.

## Refleksija

- [ ] Zapiši šta ti je najviše pomoglo: tokeni ili ad-hoc vrednosti?
- [ ] Šta bi pomerio u globalne media queries sledeći put?
